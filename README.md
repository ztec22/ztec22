## Proyectos

- [AI-Agent](https://github.com/ztec22/AI-Agent): Agente de IA con Tools y Skills capaz de navegar el sistema de ficheros, crear archivos y leerlos. Con Python, LangChain y LMStudio.
- [patient-health](https://github.com/ztec22/patient-health): Microservicio API RESTful con Python & FastApi, Pydantic v2 / SqlAlchemy / PostgreSQL, Tests con Pytest & Testcontainers, Swagger / OpenAPI, Docker, CI/CD con Github Actions
- [ProductInventory](https://github.com/ztec22/ProductInventory): Microservicio API RESTful con .NET 10, ASP.NET Web Api, EntityFramework Core / PostgreSQL, Tests con Xunit, Swagger / OpenAPI, Docker, CI/CD con Github Actions
- [Fichaje-SpringBoot](https://github.com/ztec22/Fichaje-SpringBoot): Microservicio API RESTful con MongoDB, Arquitectura Hexagonal, TestContainers, Docker compose, Kubernetes, CI/CD con Github Actions
- [Kafka-SpringBoot](https://github.com/ztec22/Kafka-SpringBoot):Microservicios que se comunican mediante la cola de mensajería Apache Kafka. Monitorización con ELK Stack (ElasticSearch, Logstash, Kibana) 
- [RabbitMQ-SpringBoot](https://github.com/ztec22/RabbitMQ-SpringBoot): Microservicios que se comunican mediante la cola de mensajería RabbitMQ. Monitorización con Grafana y Prometheus

- [job_applications](https://github.com/ztec22/job_applications): Aplicación web MVC con Elixir y Phoenix para la gestión de solicitudes de ofertas de trabajo. Docker compose, CI/CD con Github Actions
